# 메이플스토리 서비스

## 프로젝트 개요
**우아한테크코스 4주차 오픈미션**  
- 프리코스 경험 기반으로 관련된 분야의 미션을 설계하고 구현한다.
- 스스로 도전하고 싶은 목표를 설정하고, 그에 맞는 실행 계획을 세워 미션을 진행한다.
- 메이플스토리의 open API를 이용해, 자신의 캐릭터 정보를 조회하는 애플리케이션 구현

## 목표
- 본인의 캐릭터 정보를 조회하는 기능 구현
- 추후 추가 예정

## 구현 기능 목록
- [x] 넥슨 Open API를 이용해 내 캐릭터 정보 조회
  - [x] 닉네임을 입력받아 캐릭터 고유 ID 조회
  - [x] 캐릭터 기본 정보(직업, 레벨, 월드명 등) 출력
- [x] 예외 상황 발생 시 사용자에게 에러 메시지 출력

## 고려 예외 사항
- [x] 존재하지 않는 캐릭터명을 입력한 경우
- [ ] API 호출 실패 (서버 오류, 응답 지연 등)
- [ ] API Key가 유효하지 않거나 누락된 경우

## 환경 변수 설정
`application.yml`은 GitHub에 업로드하지 않습니다.  
로컬 환경에서 아래 명령어로 API 키를 설정해야 합니다.

**Windows**
