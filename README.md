# 메이플스토리 서비스

## 프로젝트 개요
**우아한테크코스 4주차 오픈미션**  
- 프리코스 경험 기반으로 관련된 분야의 미션을 설계하고 구현한다.
- 스스로 도전하고 싶은 목표를 설정하고, 그에 맞는 실행 계획을 세워 미션을 진행한다.
- 메이플스토리의 open API를 이용해 메이플스토리 관련 서비스 제공

## 목표
- 본인의 캐릭터 정보를 조회하는 기능 구현
- 큐브 시뮬레이션 기능 구현
- 강화 시뮬레이션 기능 구현
- 뽑기 시뮬레이션 기능 구현
- 추후 추가 예정

## 고려 예외 사항
- [x] 존재하지 않는 캐릭터명을 입력한 경우
- [ ] API 호출 실패 (서버 오류, 응답 지연 등)
- [ ] API Key가 유효하지 않거나 누락된 경우
- [ ] 잘못된 장비 가격을 입력한 경우
  - [ ] 음수
  - [ ] 숫자가 아님
  - [ ] 너무 큰 수
