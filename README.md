# 메이플스토리 서비스

## 프로젝트 개요
**우아한테크코스 4주차 오픈미션**  
- 프리코스 경험 기반으로 관련된 분야의 미션을 설계하고 구현한다.
- 스스로 도전하고 싶은 목표를 설정하고, 그에 맞는 실행 계획을 세워 미션을 진행한다.
- 메이플스토리의 open API를 이용해 메이플스토리 관련 서비스 제공

## 목표
- 본인의 캐릭터 정보를 조회하는 기능 구현
- 큐브 시뮬레이션 기능 구현
- 강화 시뮬레이션 기능 구현
- 뽑기 시뮬레이션 기능 구현
- 추후 추가 예정

- [x] 넥슨 Open API를 이용해 내 캐릭터 정보 조회

  - [x] 예외 상황 발생 시 사용자에게 에러 메시지 출력
- [ ] 큐브 시뮬레이션
  - [x] 잠재능력 재설정 기능
    - [x] 랜덤으로 옵션을 뽑는 기능

    - [x] 확률적으로 등업하는 기능
    - [x] 천장 스택이 다 쌓이면 다음엔 100% 등업하는 기능   
  - [x] 에디셔널 잠재능력 재설정 기능
    - [x] 랜덤으로 옵션을 뽑는 기능

  - [x] 초기화 기능
  - [ ] 통계 출력 기능
    - [ ] 잠재능력 재설정 횟수

  - [ ] 초기 설정 기능

## 고려 예외 사항
- [x] 존재하지 않는 캐릭터명을 입력한 경우
- [ ] API 호출 실패 (서버 오류, 응답 지연 등)
- [ ] API Key가 유효하지 않거나 누락된 경우
